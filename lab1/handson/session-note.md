# 세션 시작 노트

## 세션 목표

- 다음 액션을 스스로 고를 수 있는 작은 Gemini 에이전트를 만듭니다.
- 최종 결과물을 쓰기 전에 로컬 마크다운 노트를 먼저 읽습니다.
- 진행자를 위한 짧은 브리프를 마크다운 파일로 저장합니다.

## 작성 규칙

- 브리프는 1분 안에 읽을 수 있을 만큼 짧게 작성합니다.
- 세 가지 세션 목표를 직접 언급합니다.
- 마지막 줄에는 진행자가 가장 먼저 해야 할 일을 한 문장으로 적습니다.
